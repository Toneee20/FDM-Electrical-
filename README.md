# FMD Electrical (Pty) Ltd — Website

Professional single-page application for FMD Electrical.

## Deployment

### Vercel (Recommended)
1. Push this repository to GitHub
2. Import the repository into [Vercel](https://vercel.com)
3. Vercel auto-detects the static site — no build command needed
4. Click **Deploy**

### GitHub Pages
1. Push to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Site will be live at `https://yourusername.github.io/repo-name`

## Files
- `index.html` — Full website (self-contained, all assets embedded)
- `vercel.json` — Vercel SPA routing config
